# Algo Automation

This project automates the creation and deletion of AvodMD algorithms using TypeScript.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create a `.env` file with your credentials:
   ```
   EMAIL=your-email
   PASSWORD=your-password
   ```

3. Run the script:
   ```bash
   # To create an algo:
   npm run create

   # To delete an algo:
   npm run delete <algoId>
   ```
